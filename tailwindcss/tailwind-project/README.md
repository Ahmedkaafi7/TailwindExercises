# Birmad Security – Tailwind Project

This is a responsive landing page built using **HTML, Tailwind CSS (CDN), and a little JavaScript**.

## 🚀 About the Project

Birmad Security is a simple business website with:

* Responsive Navbar (with mobile menu toggle)
* Hero Section with gradient background
* Features Section
* Testimonials
* Pricing Plans
* Contact Form
* Footer

I built this project using **only Tailwind utility classes** without writing custom CSS.

## 🛠️ Technologies Used

* HTML5
* Tailwind CSS (CDN)
* JavaScript (for mobile menu toggle)

## 📚 What I Practiced

* Tailwind layout (Flexbox & Grid)
* Responsive design (`md:` breakpoints)
* Hover effects and transitions
* Building clean UI sections
* Mobile-first approach

## 🎯 Goal

The goal of this project was to improve my Tailwind CSS skills and understand how to structure a professional landing page.
